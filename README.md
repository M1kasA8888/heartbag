## 📋 项目简介

本项目是一个功能完整的无人机航线规划与飞行监控系统，实现了从障碍区设置、航线规划到实时飞行监控的完整工作流程。系统采用交互式地图界面，支持障碍区圈选、智能避障路径规划、安全半径设置和实时飞行数据监控。

## ✨ 核心功能

### 1. 🗺️ 地图交互与障碍区管理
- **交互式地图**：基于 Folium 的高交互性地图界面
- **障碍区圈选**：支持在地图上绘制多边形障碍区
- **障碍区管理**：查看、删除已保存的障碍区
- **可视化显示**：红色半透明区域显示障碍区范围

### 2. 🎯 航线规划系统
- **智能起降点设置**：点击地图设置起飞点（绿色）和目标点（红色）
- **自动避障算法**：基于 Shapely 几何计算的路径规划
- **安全半径控制**：可调节的无人机与障碍物安全距离（10-100米）
- **路径优化**：直线优先，遇障自动生成绕行路径
- **航线信息**：显示总航程、预计飞行时间、航点数量

### 3. 🎮 实时飞行监控
- **实时位置追踪**：地图上实时显示无人机位置
- **多维度监控**：
  - 当前航点/总航点
  - 实时飞行速度
  - 已用时间/预计到达时间
  - 剩余距离/完成进度
  - 电池电量模拟
  - 安全状态检测
- **动态图表**：实时显示高度、速度、剩余距离变化曲线
- **飞行控制**：支持手动终止飞行
- **飞行总结**：任务完成后自动生成统计报告

### 4. 📊 数据可视化
- 实时飞行数据图表
- 航段进度跟踪
- 飞行历史数据记录
- 完成后自动生成数据总结
